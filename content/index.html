Background Colors

  • .bg-color0 - Default White #fff (default body bg-color)
  • .bg-color1 - Default Gray #666 (table headers)
  • .bg-color2 - Light Gray #efefef (Form fields and table rows)
  • .bg-color3 - Pattern for the header and footer
  • .bg-color4 - Dark gray #333 used on .work-box mod

Text Colors

  • .color0 - White #fff
  • .color1 - Gray, used as body default. #666
  • .color2 - Very Dark Gray - Used for H1s #333
  • .color3 - Dark Blue, used for links, decretive quotes, and default footer color #0473b5
  • .color4 - Light Blue, used for links that are hovered or have been visited #64a9d2
  • .color5 - Very Light Gray - Used inside form items, like comments, to define the text as editable by users #aaa
  • .color6 - Cyan #00aeef - used for link hovers
